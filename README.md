# imageCapture

capturing still images from your computer's camera, and delivering it to your server

# 使用说明

+ put all code into Tomcat
+ config tomcat server.xml -> <Context path="" docBase="E:\tomcat6\webapps\cam1" debug="0" reloadable="true" crossContext="true" />
+ run `localhost:8080/test.html`
